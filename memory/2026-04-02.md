### 关键发现

1. **FrePPLe 数据库隔离：** Web UI 连的是 `frepple` 主库，不是 `frepple0~30` 场景库
2. **排程异常：** Web Service 启动排程任务时抛出异常（`Exception launching task`），可能与 Celery 后台队列有关
3. **Excel 导入入口：** 任务状态页面右侧面板有"导入Excel"和"导出Excel"功能
4. **API 可用：** `/api/input/{object}/` 支持 GET/POST/PUT/DELETE
5. **采购报告 "Unknown supplier"：** 是因为 Item 的 procurement type 配置错误，而非真的缺少供应商
