### 工序体系（15种）
切割 → 倒边 → 铣磨 → 磨砂 → 封蜡上盘 → 下盘 → 光胶上盘 → 粗抛 → 精抛 → 清洗 → 检验光洁度 → 检验角度 → 检验材料 → 涂漆 → 封蜡上夹具

**加工模式:**
- 自动运行: 机器自动加工，人员仅上下机（铣磨、切割、粗抛、精抛）
- 人机协同: 人员和机器同时运作，不能离开（单轴机磨砂/倒边）
- 人员单独: 纯辅助工序（封蜡、下盘、清洗、检验等）

### 行业知识摘要
- **行业标准:** GB/T 7660《光学零件冷加工规范》、GB/T 1185《光学零件表面疵病》、GB/T 2831《光学零件面形精度》
- **三技术阶段:** 传统冷加工→数控加工→超精密加工(磁流变/离子束)
- **三梯队企业:** 舜宇/凤凰/中光学(规模龙头)→蓝特/波长/宇迪(专精特新)→中小定制化厂(拍频+鑫望)
- **市场规模:** 2025年中国超800亿元，全球2026年268亿元(CAGR 14%)
- **核心痛点(6维度):** 订单模式/工艺管理/资源调度/计划排程/供应链库存/质量追溯
- **PBS棱镜工艺路线示例:** 切割(0.2h)→粗磨(0.5h)→精磨(0.8h)→抛光(4h)→镀膜(8h)→胶合(1h)→全检(0.5h)

### FrePPLe技术架构摘要
- **算法:** 层级式反向展开 + 约束传播
- **部署:** Docker容器化(推荐)/二进制包/K8s
- **扩展:** 自定义属性(registerAttribute)→Python钩子(5种)→Django App
- **Python钩子类型:** 排程前/资源分配/约束检查/排程后/异常处理
